# ktcp:model

## 概要

KTCP（Keruta Task Client Protocol）プロトコルのデータモデルを定義するモジュール。

## 責務

- プロトコルメッセージのデータモデル定義
- タスク状態のエンティティ定義
- バリデーションルール定義
- シリアライズ/デシリアライズインターフェース

## メッセージタイプ

メッセージタイプの詳細は [KTCP 仕様](../task-client-protocol.md) を参照してください。

### タスク操作（CRUD）
- タスク作成要求/応答
- タスク情報取得要求/応答
- タスク更新要求/応答
- タスク削除要求/応答
- タスク一覧取得要求/応答

### タスク実行
- タスク実行要求
- タスク状態更新
- タスク完了通知
- タスクキャンセル要求

### その他
- 認証メッセージ
- ハートビート
- ログストリーミング
- エラー通知

## 依存関係

外部モジュールに依存しない
